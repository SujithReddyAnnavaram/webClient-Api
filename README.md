# WebClient API — Spring Boot

A Spring Boot application demonstrating how to consume 
external REST APIs using Spring WebFlux WebClient.

## Tech Stack
- Java 17
- Spring Boot
- Spring WebFlux
- WebClient (reactive HTTP client)
- Maven

## Features
- Consumes external REST APIs reactively
- Built with WebClient instead of RestTemplate
- Demonstrates modern reactive programming approach

## How to Run
1. Clone the repository
2. Run the Spring Boot application
3. Access via `http://localhost:8080`
